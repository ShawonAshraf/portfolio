## Using as your portfolio
Edit the `information.json` and `utils.ts` files as you require. Make sure to update the commands in `App.tsx`
accordingly.

## Running locally
- Clone the repo
- `cd` into the directory
- Then run

```bash
# dev mode with hot reload
yarn run start

# important, run tests
yarn run test

# create a production build
yarn run build
```
